 1) data
    ->data/test
    ->data/train
    test_labels.csv
    train_labels.csv

2) models
(folder for storing saved models, as .h5 files)

3) metrics
   ->metrics/plots
   ->metrics/reports
(folder for storing saved plots, reports etc)
